# Genii ERP DIY Scaffold ($47)

Welcome to the autonomous frontier. This package contains the blueprints to build your own Genii core.

## 1. Database Schema (SQLAlchemy)
The raw models are located in `backend/models/database.py`. You can export these to any PostgreSQL or SQLite instance.

## 2. Obsidian Brain Structure
Follow this folder hierarchy in your vault:
- `Brain/`
  - `Context/` (Long-term memory)
  - `Active/` (Current tasks)
  - `Synthesis/` (Agent reports)

## 3. OpenClaw Basic Config
Map your local LLM (LM Studio/Ollama) to the `openclaw.json` provided in this zip.
